About

This is the Henry theme for the Zola static site generator. Henry is based on the original styles of Jekyll, and the name is the first name of Dr. Henry Jekyll.

The theme is licensed under the MIT license, so you may freely use and redistribute either for commercial or non-commercial work.

You can find the source code for the theme in the sirodoht/zola-henry repository. For a description of this theme and links to other Zola themes, please go to https://www.getzola.org/themes/.
